### INSTALLATION ###

1) Open terminal and navigate to the project folder
2) run the following command : npm install
3) start a mysql server and import the database with the "database.sql" file
4) edit the variables in ".env" file to match your database
5) run the application with the "startapp.bat" file or by typing "npm run start"

### TESTING ###

To get the testing report, you can run the command "npm run test" or run the "tests.bat" file.
